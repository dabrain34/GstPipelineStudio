🎉 New Release Announcement: GstPipelineStudio v0.5.1 🎉

It's a great pleasure to announce the release of GstPipelineStudio version 0.5.1! The app has been renamed to dev.mooday.GstPipelineStudio and brings major new features including remote pipeline introspection, crash recovery, DOT file loading, auto-connect on node click, auto-arrange layout, and improved Flathub metadata compliance.

Highlights from 0.5.0 and 0.5.1:

- Remote pipeline introspection
- DOT file loading
- Auto-connect on element click
- Auto-arrange pipeline layout
- Semi-transparent node backgrounds and dynamic node width
- App ID renamed to dev.mooday.GstPipelineStudio
- GStreamer 1.28.0
- GTK 4.20.3
- RPM packaging for Fedora
